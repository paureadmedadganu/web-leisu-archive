# web-leisu-archive

## 项目简介

`web-leisu-archive` 是一个用于归档与发布多个独立 HTML 页面的通用仓库。本仓库不针对任何特定网站或域名，仅作为静态页面的集中存储与展示空间，适合存放各类无需后端支持的网页内容。

## 目录结构

```
web-leisu-archive/
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放归档的 HTML 页面
│   ├── example.html
│   └── ...
├── assets/             # 页面引用的静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

## 页面归档说明

- 所有归档的 HTML 页面存放在 `pages/` 目录下，每个文件应保持独立完整，避免跨文件依赖。
- 页面可引用 `assets/` 目录下的公共资源，建议使用相对路径。
- 如需新增归档页面，可直接在 `pages/` 下添加 HTML 文件，并更新入口索引（如有）。
- 归档内容应遵守相关法律法规，不得包含侵权、违法或不当信息。

## 维护说明

- 本仓库由维护者不定期更新，用于整理和发布可公开访问的静态页面。
- 欢迎提交 Issue 或 Pull Request 来建议新增或修正页面内容，但请注意保持内容的中立与合规。
- 对于不再需要的归档页面，维护者会及时清理或移至历史目录。

## 注意事项

- 本仓库仅作为技术归档用途，不保证页面内容长期可用或更新。
- 页面中不应包含任何形式的跟踪、广告或恶意脚本。
- 使用本仓库内容所产生的任何后果由使用者自行承担。
